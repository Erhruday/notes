# Uso de un input tipo hidden requerido

No se puede.
Usamos un input de tipo texto en su lugar y lo ocultamos.

> Estilos:

    .hidden-input {
        float   : left; /**
        opacity : 0;
        width   : 0;
    }


> HTML:

    <input type="text" class="hidden-input" required>
